# 论文PDF阅读器：工具栏功能详解

阅读器的顶部工具栏是操作的核心区域，包含查看、视图切换、AI辅助和文件管理四大功能组。熟练掌握工具栏的各个按钮，可以大幅提升阅读效率。

---

## 工具栏布局

工具栏从左到右分为四个功能组，每个组的按钮都有明确的图标和提示文字：

1. **查看组**：缩放和查找功能
2. **视图组**：三种视图模式切换
3. **AI辅助组**：五个智能辅助功能
4. **文件组**：笔记、保存、打印等操作

---

## 查看组功能

### 放大（Zoom In）

- **图标**：放大镜加号
- **功能**：增大PDF页面的显示比例，看清细节内容。
- **使用场景**：查看公式、图表、小字号文本等细节。
- **快捷键**：Ctrl + +（加号）

### 缩小（Zoom Out）

- **图标**：放大镜减号
- **功能**：减小PDF页面的显示比例，查看更多内容。
- **使用场景**：浏览页面整体布局、快速翻阅文档。
- **快捷键**：Ctrl + -（减号）

### 查找（Find）

- **图标**：搜索放大镜
- **功能**：在当前文档中搜索关键词。
- **可用模式**：**仅在单视图模式下可用**（原文或译文视图）。
- **不可用原因**：中英对照视图下查找功能被禁用，因为双列同步滚动会导致搜索结果定位复杂。
- **使用方法**：
  1. 切换到单视图模式。
  2. 点击查找按钮，顶部会出现搜索框。
  3. 输入关键词，系统会高亮显示所有匹配项。
  4. 使用「上一个」和「下一个」按钮在匹配项间跳转。

## 视图组功能

视图组包含三个按钮，用于在三种视图模式间切换：

### 中英对照

- **图标**：翻译符号（两个箭头交叉）
- **功能**：进入双视图模式，左右分屏显示原文和译文。
- **状态**：始终可用（除非文档未开始翻译）。
- **提示**：这是默认视图模式，适合初次阅读。

### 原文

- **图标**：英文字母标识
- **功能**：进入英文原文单视图模式。
- **状态**：翻译完成后可用（翻译进行中为灰色禁用状态）。
- **提示**：在此模式下可以选中文本添加笔记。

### 译文

- **图标**：中文字符标识
- **功能**：进入中文译文单视图模式。
- **状态**：翻译完成后可用（翻译进行中为灰色禁用状态）。
- **提示**：适合快速浏览和理解论文大意。

**重要提示**：原文和译文按钮在翻译进度未达到100%时会显示为灰色，无法点击。请等待翻译完成后才能切换到单视图模式。

## AI辅助组功能

AI辅助组提供五个强大的智能功能，帮助您深入理解和分析论文：

### 问答（Chat）

- **图标**：机器人头像
- **功能**：与AI助手对话，提问关于论文的问题。
- **使用场景**：
  - 询问专业术语的含义。
  - 请求解释复杂的概念或方法。
  - 讨论论文的研究思路和创新点。
- **特点**：AI已通读整篇论文，可以回答上下文相关的问题。

### 思维导图（Mind Map）

- **图标**：思维导图符号
- **功能**：自动生成论文的思维导图，展示文章结构和逻辑关系。
- **使用场景**：
  - 快速把握论文的整体框架。
  - 理清研究思路和论证脉络。
  - 制作学习笔记和复习材料。
- **特点**：可视化呈现，支持展开/折叠节点。

### 论文导读（Note）

- **图标**：报告文档符号
- **功能**：生成结构化的论文导读报告，包括摘要、研究背景、方法、核心发现、创新点、局限性等。
- **使用场景**：
  - 快速了解论文核心内容。
  - 撰写文献综述时提取关键信息。
  - 对比多篇论文的研究方法。
- **特点**：内容结构化，便于快速检索和理解。

### 提取公式（Formula）

- **图标**：数学公式符号
- **功能**：识别并提取论文中的数学公式，转换为可编辑格式。
- **使用场景**：
  - 复制公式到自己的文档中。
  - 学习公式的推导过程。
  - 验证公式的正确性。
- **特点**：支持LaTeX格式输出，方便学术写作。

### 提取表格（Table）

- **图标**：表格符号
- **功能**：识别并提取论文中的数据表格，转换为可编辑的Excel或CSV格式。
- **使用场景**：
  - 复用论文中的实验数据。
  - 进行数据分析和对比。
  - 制作自己的研究表格。
- **特点**：保持表格结构和格式，支持导出。

**注意**：所有AI辅助功能都需要消耗一定的AI额度。如果额度不足，系统会提示您购买资源包。

## 文件组功能

文件组提供文档管理和导出功能：

### 阅读笔记

- **图标**：笔记本符号
- **功能**：打开阅读笔记侧边栏，查看和管理当前文档的所有笔记。
- **使用场景**：
  - 回顾之前添加的笔记。
  - 编辑或删除已有笔记。
  - 复制笔记内容到其他软件。
- **特点**：笔记按页码排序，点击可跳转到对应位置。

### 保存（Download）

- **图标**：下载箭头
- **功能**：下载翻译后的PDF文档。
- **可用模式**：**仅在单视图模式下可用**。
- **不可用原因**：中英对照视图是实时渲染的双列布局，无法直接保存为PDF文件。
- **使用方法**：
  1. 切换到原文或译文单视图模式。
  2. 点击保存按钮。
  3. 选择保存路径，下载当前视图的PDF。

### 打印（Print）

- **图标**：打印机
- **功能**：打印当前PDF文档。
- **可用模式**：**仅在单视图模式下可用**。
- **不可用原因**：同保存功能，双视图模式无法直接打印。
- **使用方法**：
  1. 切换到单视图模式。
  2. 点击打印按钮。
  3. 选择打印机和打印设置。

## 工具栏使用技巧

### 1. 悬停查看提示

将鼠标悬停在任意按钮上，会显示该功能的简短说明，帮助记忆各按钮的作用。

### 2. 分组理解功能

工具栏按功能分组，可以快速定位需要的功能：
- 调整显示 → 查看组
- 切换语言 → 视图组
- 智能分析 → AI辅助组
- 文件操作 → 文件组

### 3. 注意按钮状态

- **正常状态**：黑色图标，可以点击使用。
- **禁用状态**：灰色图标，当前不可用（如翻译未完成时的原文/译文按钮）。
- **选中状态**：蓝色高亮，表示当前激活的模式（如当前的视图模式）。

### 4. 常用功能快捷访问

- **频繁切换视图**：使用 Ctrl + Tab 快捷键，比点击按钮更快。
- **快速放大缩小**：使用 Ctrl + 滚轮，或 Ctrl +/- 快捷键。
- **AI功能按需使用**：不需要每次都打开所有AI面板，用时再点击对应按钮。

## 常见问题

### 为什么有些按钮是灰色的？

灰色表示该功能当前不可用，常见原因：
- 翻译未完成：原文和译文按钮需要等待翻译100%完成。
- 视图限制：查找、保存、打印在双视图模式下不可用。

### AI辅助功能需要联网吗？

是的，AI辅助功能需要连接服务器进行处理，请确保网络连接正常。

### 如何关闭已打开的AI面板？

再次点击对应的AI按钮，或点击面板右上角的关闭按钮（X）。
