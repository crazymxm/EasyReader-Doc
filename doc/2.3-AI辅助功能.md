# 论文PDF阅读器：AI 辅助功能

本软件集成了五种强大的AI辅助功能，帮助您深入理解、分析和利用学术论文。这些功能基于先进的AI技术，能够自动提取、总结和生成内容，大幅提升科研效率。

---

## AI 辅助功能概览

在论文PDF阅读器工具栏的「AI辅助」组中，有五个功能按钮：

1. **问答**：与AI对话，提问论文相关问题
2. **思维导图**：自动生成论文结构导图
3. **论文导读**：生成结构化导读报告
4. **提取公式**：识别并提取数学公式
5. **提取表格**：识别并提取数据表格

所有AI功能都会在右侧打开独立面板，不影响PDF阅读区域（单视图模式下PDF区域会自动缩窄以适应AI面板）。

---

## 1. 问答功能（Chat）

### 功能说明

问答功能提供一个智能对话界面，AI助手已经通读了整篇论文，可以回答您关于论文的任何问题。

### 使用方法

1. 点击工具栏的 **「问答」** 按钮（机器人图标）。
2. 右侧会打开对话面板。
3. 在输入框中输入您的问题。
4. AI会基于论文内容给出回答。
5. 可以继续追问或提出新问题。

### 使用场景

- **术语解释**：询问专业术语的含义和背景。
- **方法理解**：请求解释研究方法的原理和步骤。
- **结果分析**：讨论实验结果的意义和影响。
- **创新点探讨**：了解论文的创新之处和贡献。
- **局限性分析**：询问研究的不足之处。

### 示例问题

- "这篇论文的研究方法是什么？"
- "请解释一下XXX术语的含义"
- "这个实验的主要发现有哪些？"
- "论文的创新点在哪里？"
- "这项研究有什么局限性？"

### 注意事项

- AI的回答基于论文内容，可能不包含外部知识。
- 对于复杂问题，可以尝试分解为多个小问题。
- 对话历史会在当前会话中保留，关闭面板后清空。

## 2. 思维导图（Mind Map）

### 功能说明

思维导图功能自动分析论文结构，生成可视化的层级导图，展示论文的逻辑框架和内容组织。

### 使用方法

1. 点击工具栏的 **「思维导图」** 按钮。
2. 系统开始分析论文结构（需要几秒到几十秒，取决于论文长度）。
3. 生成的思维导图显示在右侧面板中。
4. 可以展开/折叠各个节点查看详情。
5. 点击节点可以快速跳转到对应章节。

### 思维导图结构

典型的论文思维导图包括以下层级：

- **根节点**：论文标题
  - **摘要**：研究概述
  - **引言**：研究背景和目的
    - 研究背景
    - 研究问题
    - 研究目标
  - **方法**：研究方法和技术
    - 数据来源
    - 实验设计
    - 分析方法
  - **结果**：主要研究发现
    - 实验结果1
    - 实验结果2
    - ...
  - **讨论**：结果分析和解释
  - **结论**：总结和未来展望
  - **参考文献**：引用的文献列表

### 使用场景

- **快速把握结构**：初次接触论文时，通过导图了解整体框架。
- **定位特定章节**：快速找到感兴趣的章节内容。
- **制作学习笔记**：将导图作为复习和记忆的辅助工具。
- **对比多篇论文**：为不同论文生成导图，比较其结构差异。

### 注意事项

- 生成思维导图需要消耗AI额度。
- 复杂论文的导图可能较为庞大，建议展开关键节点查看。
- 导图是基于论文大纲和标题生成的，可能与实际内容略有偏差。

## 3. 论文导读（Note）

### 功能说明

论文导读功能生成一份结构化的详细报告，系统地总结论文的核心内容，包括研究背景、方法、发现、创新点等关键信息。

### 使用方法

1. 点击工具栏的 **「论文导读」** 按钮。
2. 系统开始分析论文并生成导读（可能需要较长时间）。
3. 生成的导读报告显示在右侧面板中。
4. 可以滚动浏览各个部分的内容。
5. 点击「复制」按钮可以将导读内容复制到剪贴板。

### 导读报告结构

导读报告包含以下部分：

#### 1. 论文概览
- **摘要**：论文的完整摘要
- **关键词**：核心关键词列表
- **术语解释**：重要专业术语的解释

#### 2. 研究问题与方法
- **研究背景**：研究的动机和背景信息
- **研究目的**：要解决的核心问题
- **研究方法**：采用的技术和方法

#### 3. 核心发现
- **数据/实验设置**：实验设计和数据来源
- **主要结论**：关键研究发现和结果

#### 4. 价值、局限与展望
- **创新点**：论文的创新之处
- **意义与启示**：研究的理论和实践意义
- **局限性**：研究的不足之处
- **未来展望**：后续研究方向

### 使用场景

- **快速了解论文**：无需阅读全文，通过导读掌握核心内容。
- **文献综述写作**：提取关键信息用于综述文章。
- **多论文对比**：为多篇论文生成导读，进行横向比较。
- **汇报准备**：使用导读内容制作学术报告或演示文稿。

### 导出功能

导读报告支持导出为多种格式：

- **TXT格式**：纯文本，适合快速查看
- **Markdown格式**：结构化文本，适合编辑和分享
- **Word格式**：可直接在Microsoft Word中打开和编辑

### 注意事项

- 生成导读需要较多时间，请耐心等待。
- 导读质量依赖于论文的清晰度和结构完整性。
- 某些专业领域的术语可能需要人工校对。

## 4. 提取公式（Formula）

### 功能说明

提取公式功能使用OCR和AI技术识别论文中的数学公式，并将其转换为可编辑的LaTeX代码或其他格式。

### 使用方法

1. 点击工具栏的 **「提取公式」** 按钮。
2. 系统扫描全文，识别所有数学公式。
3. 识别结果以列表形式显示在右侧面板。
4. 每个公式显示原始图片和LaTeX代码。
5. 点击「复制」按钮可以复制LaTeX代码。

### 输出格式

- **LaTeX代码**：标准的数学公式标记语言，可在LaTeX编辑器中使用。
- **图片**：公式的原始截图，用于参考。

### 使用场景

- **学术写作**：在自己的论文中复用公式。
- **学习推导**：研究公式的构成和推导过程。
- **公式验证**：检查公式的正确性和完整性。
- **教学材料**：制作课件和教材时使用标准公式。

### 注意事项

- 复杂公式的识别准确率可能不高，需要人工校对。
- 手写公式或低质量扫描的公式识别效果较差。
- 提取公式功能需要消耗AI额度。

## 5. 提取表格（Table）

### 功能说明

提取表格功能识别论文中的数据表格，并将其转换为可编辑的结构化数据格式（如Excel、CSV）。

### 使用方法

1. 点击工具栏的 **「提取表格」** 按钮。
2. 系统扫描全文，识别所有表格。
3. 识别结果以列表形式显示在右侧面板。
4. 每个表格显示预览和数据结构。
5. 点击「导出」按钮可以选择导出格式。

### 输出格式

- **Excel格式**：.xlsx文件，可在Microsoft Excel中打开和编辑。
- **CSV格式**：逗号分隔值文件，适合数据处理和分析。
- **HTML表格**：网页格式的表格，便于嵌入网页。

### 使用场景

- **数据复用**：在自己的研究中复用已有数据。
- **数据分析**：导入统计软件进行分析。
- **数据对比**：整合多篇论文的数据进行横向比较。
- **元分析**：收集多个研究的数据进行meta分析。

### 注意事项

- 复杂表格（合并单元格、多层表头）的识别可能不准确。
- 表格中的特殊符号和单位可能需要手动调整。
- 提取表格功能需要消耗AI额度。

## AI 功能的共同特点

### 1. 额度消耗

所有AI功能都需要消耗AI辅助额度：

- **轻度使用**：问答功能每次对话消耗较少额度。
- **中度使用**：思维导图、提取公式、提取表格消耗中等额度。
- **重度使用**：论文导读生成消耗较多额度。

您可以在顶栏查看剩余额度，额度不足时需要购买资源包。

### 2. 处理时间

AI功能的处理时间因功能和论文复杂度而异：

- **快速响应**（几秒）：问答、提取公式、提取表格
- **中等时间**（几十秒）：思维导图
- **较长时间**（几分钟）：论文导读

处理过程中会显示加载动画，请耐心等待。

### 3. 面板管理

- **同时打开**：可以同时打开多个AI面板，它们会层叠显示。
- **切换查看**：点击不同的AI按钮可以在各面板间切换。
- **关闭面板**：点击面板右上角的X按钮或再次点击工具栏按钮关闭。

### 4. 网络依赖

所有AI功能都需要联网使用：

- 确保网络连接正常。
- 网络不稳定可能导致请求失败或响应缓慢。
- 建议在稳定的网络环境下使用AI功能。

## 使用建议

### 高效使用流程

1. **初次阅读**：先使用「论文导读」快速了解全文概要。
2. **结构把握**：查看「思维导图」理清论文逻辑框架。
3. **深入阅读**：在阅读过程中使用「问答」解决疑问。
4. **数据提取**：根据需要「提取公式」和「提取表格」。
5. **笔记整理**：结合AI生成的内容添加个人笔记。

### 节省额度技巧

- 优先使用免费的阅读和笔记功能。
- AI功能按需使用，避免重复生成相同内容。
- 问答功能尽量提出具体问题，避免泛泛而谈。
- 对于简单问题，可以先尝试自己查找答案。

### 常见问题

#### AI功能无法使用怎么办？

- 检查网络连接是否正常。
- 确认账号已登录且有足够额度。
- 尝试刷新页面或重启软件。
- 如仍无法使用，联系客服寻求帮助。

#### AI生成的内容不准确怎么办？

- AI功能基于算法生成，可能存在误差。
- 建议将AI输出作为参考，人工校对重要内容。
- 对于关键数据和方法，请以原文为准。
- 可以向开发团队反馈错误，帮助改进算法。

#### 如何查看AI额度的使用情况？

- 点击顶栏的用户头像，查看会员等级和额度信息。
- 进入「设置」→「我的资源包」查看详细的使用记录。
- 系统会在额度即将用尽时发出提醒。
